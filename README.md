## 🚀 Running Locally

```sh-session
git clone https://github.com/Panthu13147/myportfolio.git
cd myportfolio
```

Create a `.env` file similar to [`.env.example`]([https://github.com/Panthu13147/myportfolio/blob/main/.env.example](https://github.com/Panthu13147/myportfolio/blob/main/.env.example)) to connect the necessary database/APIs.

Then install dependencies and run the development server:
```sh-session
pnpm install
pnpm dev
```

## ⚙️ The Architecture

* **Frontend:** Responsive UI using Next.js 13+ App Router and TailwindCSS.
* **Backend:** Python API handling form submissions and logic.
* **Database:** Real-time data persistence using PostgreSQL.
